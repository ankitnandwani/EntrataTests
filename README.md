This is selenium java project done as part of Entrata interview process.

### System requirements
- JDK 8 or above


### Steps to clone and execute the tests

```
git clone https://github.com/ankitnandwani/EntrataTests.git
```
```
mvn verify 
```


### Libraries Used
- selenium-java
- junit
- webdrivermanager
- owner
- javafaker
